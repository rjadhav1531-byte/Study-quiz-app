# Study Quiz Maker

A simple web app that turns PDF or image notes into a quiz.

## Features
- Upload `.pdf` or image files.
- Extract text from PDF pages (`pdf.js`).
- Extract text from images with OCR (`Tesseract.js`).
- Generate multiple-choice and true/false practice questions.
- Instant answer feedback and final score.

## Run
1. Open `index.html` in a browser with internet access.
2. Upload a PDF or image.
3. Click **Extract Text**.
4. Click **Generate Quiz**.

## Notes
- OCR on large images can take time.
- This app uses CDN scripts, so internet is required.
